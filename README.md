# Shell management scripts
Various shell scripts for automating linux tasks.

# Scripts
- backup.sh - script for backuping up my configs
- docker-manager.sh - script to control docker
- nginx-manager.sh - script to control nginx
- tls-manager.sh - script to control TLS certificates via letsencrypt
- wireguard-manager.sh - script to control wireguard & users
- xray-manager.sh - script to control xray & users (also: no-web-panel subscriptions, link & qr code generation, sub secret path generation)
